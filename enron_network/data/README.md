# Data Notes

This folder contains the raw and processed data for the Enron network project.

## Raw Data

- `raw/email-Enron.txt.gz`: SNAP Enron email network edge list

Original source:

- https://snap.stanford.edu/data/email-Enron.html
- https://snap.stanford.edu/data/email-Enron.txt.gz

## Processed Data

The `processed/` folder contains exported CSV and JSON files generated by the notebook for report tables and cross-network comparison.
